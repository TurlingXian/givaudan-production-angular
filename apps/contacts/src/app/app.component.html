a simple contact page with these features

  • list the contacts(sort and filter)
  • show a contact information - hover on the id, then click
  • create a contact - click the add button; update a contact - click the edit button
  • Import a CSV - click import btn, do not have the check and filter, so please use the right format.
  • export a CSV - click export btn

Attention: do not have an overall errorcatch, if there is an error alert, please refresh the page. Please upload a right format CSV. If you have the id, it will update; if not, it will add. No check and filter, so it will add empty line and have error sometimes.
